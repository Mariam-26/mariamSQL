UNION

  • Permet de mettre vout-à-bout les résultats de plusieurs requêtes utilisant elles-même la commande SELECT .
  • Permet de concaténer les résultats de 2 requêtes ou plus.
  • CONDITION : chacune des requêtes à concaténer retournes le même nomblre de colonnes, avec les mêmes types de données et dans le même ordre.

SYNTAXE :

SELECT * FROMtable1UNION SELECT * FROM table2
