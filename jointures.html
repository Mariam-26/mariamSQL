JOINTURES

  • Sélectionner les données se trouvent dans plusieurs tables.
  • Principe : Une jointure a lieur entre deux tables. Elle exprime une correspondance entre deux clés par un ON critère d'égalité..

SYNTAXE :

SELECT * FROMtable1RIGHT JOIN table2 ON table1.id = table2.id
